مرحباً بكم في موقعي الإلكتروني الأول! (Didast-1234)

1. الموقع على GitHub Pages

هذا الموقع تم نشره بنجاح واستضافته مجاناً عبر منصة GitHub Pages ليصبح متاحاً للجميع.

2. السيرفر وعنوانه (Le serveur et son adresse)

السيرفر: يتم تشغيل هذا الموقع على خوادم شركة GitHub الضخمة تلقائياً.
العنوان: هو الرابط الذي يولده الموقع ويحتوي على اسم الحساب واسم المستودع.

3. اسم النطاق (Nom de domaine)

هو الاسم السهل الذي يكتبه البشر في المتصفح للوصول للموقع، ويقوم نظام DNS بتحويله إلى رقم IP الخاص بالسيرفر لتبسيط التصفح.
